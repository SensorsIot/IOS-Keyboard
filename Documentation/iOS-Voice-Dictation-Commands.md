# iOS Voice Dictation Commands

Complete list of voice commands for iPhone dictation (iOS 16+).

---

## Punctuation

| Say This | Result |
|----------|--------|
| Period / Full stop | . |
| Dot / Point | . |
| Comma | , |
| Question mark | ? |
| Exclamation point | ! |
| Colon | : |
| Semicolon | ; |
| Ellipsis / Dot dot dot | ... |
| Apostrophe | ' |
| Hyphen | - |
| Underscore | _ |

---

## Quotation & Brackets

| Say This | Result |
|----------|--------|
| Quote / Open quote | " |
| End quote / Close quote | " |
| Open parenthesis | ( |
| Close parenthesis | ) |
| Open bracket | [ |
| Close bracket | ] |
| Open brace | { |
| Close brace | } |

---

## Line Breaks & Spacing

| Say This | Result |
|----------|--------|
| New line | Line break |
| New paragraph | Paragraph break (double line) |
| Space bar | Add space |
| No space | No space before next word |
| No space on | Remove all spaces until "no space off" |
| No space off | Resume normal spacing |

---

## Capitalization

| Say This | Result |
|----------|--------|
| Cap | Capitalize next word |
| Caps on | Start capitalizing |
| Caps off | Stop capitalizing |
| All caps | Next word ALL CAPS |
| All caps on | Start ALL CAPS |
| All caps off | Stop ALL CAPS |
| No caps | Next word lowercase |
| No caps on | Start lowercase |
| No caps off | Stop lowercase |
